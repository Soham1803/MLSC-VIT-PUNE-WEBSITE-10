# MLSC_Website

This is the Repo for MLSC VIT-Pune Website

## Technologies Used

- ReactJS
- ChakraUI

## Installation

1. `cd frontend`
2. `npm i`
3. `npm run start`
4. Website will run on `localhost:3000`
